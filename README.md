Ogłoszenia drobne
Uncads to serwis ogłoszeń drobnych stworzony z wykorzystaniem technologii ASP.NET Core MVC oraz JavaScript. Aplikacja oferuje kompleksowy zestaw funkcjonalności takich jak: logowanie, edycję profilu, dodawanie i edycja ogłoszeń, definiowanie swoich kategorii jak i atrybutów, załączanie plików czy newsletter.
Aplikacja posiada wbudowane formularze CRUD, własne API a także system ról z panelem administratora. Dane przechowywane są w SQL Server z wykorzystaniem Entity Framework Core i ASP.NET Core Identity
